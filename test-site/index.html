    8. Browser Features

    Tabs

    Try these keyboard shortcuts:

    • Cmd/Ctrl+T -- Open new tab
    • Cmd/Ctrl+W -- Close current tab
    • Click a tab to switch, click x to close it
    • Click + to open a new blank tab

    Find in Page

    Press Cmd/Ctrl+F to open the find bar. Type a word like "button" or "test" and see matches highlighted in yellow. Press Enter to cycle through matches. Press Escape to close.

    Bookmarks

    Click the star icon (or press Cmd/Ctrl+D) to bookmark this page. The star turns gold when bookmarked. Visit pybrowser://bookmarks to see all bookmarks.

    History

    Press Cmd/Ctrl+H to open browsing history in a new tab, or navigate to pybrowser://history.

    Keyboard Shortcuts

    • Cmd/Ctrl + -- Zoom in
    • Cmd/Ctrl - -- Zoom out
    • Cmd/Ctrl 0 -- Reset zoom
    • Tab / Shift+Tab -- Cycle focusable elements
    • Up/Down arrows -- Scroll page

    9. Networking & Security

    HTTPS Indicator

    Look at the address bar:

    • HTTPS pages show a green lock icon
    • HTTP pages show a red ! warning
    • Internal pages (pybrowser://) show neither

    Try navigating to https://example.com vs http://example.com

    Gzip Compression

    The browser now sends Accept-Encoding: gzip, deflate with every request. Compressed responses are automatically decompressed. Most real websites serve gzip -- pages should load with smaller transfers now.

    CORS (Cross-Origin Resource Sharing)

    The browser enforces CORS on fetch() and XMLHttpRequest. Cross-origin requests are blocked unless the server sends Access-Control-Allow-Origin.

    Content-Security-Policy

    When a page sends a Content-Security-Policy header, the browser parses it. This helps protect against XSS attacks by restricting which scripts can run.

    WebSocket

    The browser supports the WebSocket API for real-time connections:

    var ws = new WebSocket("wss://echo.example.com");
ws.onopen = function() { ws.send("hello"); };
ws.onmessage = function(e) { console.log(e.data); };
ws.close();

    Cookies

    Cookies are automatically sent with requests and stored from Set-Cookie headers. Persisted to ~/.pybrowser/cookies.json.
